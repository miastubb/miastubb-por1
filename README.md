# miastubb-por1
# Portfolio SPA

This project is a single page application (SPA) portfolio website built using HTML, CSS, and Vanilla JavaScript.  
It showcases three core projects from the Front-End Development program and presents them in a professional portfolio format.

## Live Site

https://miastubb.github.io/miastubb-por1/

## Project Purpose

The purpose of this project is to present previous coursework in a structured and professional portfolio.  
The site demonstrates responsive layout, accessibility considerations, and modular JavaScript architecture.

## Projects Included

- RainyDays – E-commerce prototype
- JavaScript 1 – API-driven web application
- Semester Project 1 – WonderWorld science center website

Each project card includes:
- Optimized thumbnail image
- Short description
- GitHub repository link
- Live project link

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6 modules)

## Application Structure

The portfolio uses a modular JavaScript structure:

- `main.js` – application entry point
- `router.js` – hash-based SPA navigation
- `nav.js` – dynamic navigation and mobile menu
- `projects.js` – project data
- `projectCard.js` – reusable project card component

## Performance and Accessibility

The project was tested using Lighthouse to evaluate:

- performance
- accessibility
- best practices
- SEO

Performance improvements were implemented by prioritizing the hero image and optimizing images used in the project cards.

## Documents

The portfolio includes the following documents:

- CV / Resume
- Cover Letter
- Project Reflections
- Technical Project Explanation

These can be accessed directly from the **Documents** section of the site.
