# AI Usage Log

---

## Date Range

## Purpose of AI Assistance

AI tools were used as a learning assistant and reference tool for the following purposes:

- Explaining web development concepts related to HTML, CSS, and JavaScript.
- Understanding the structure and behavior of a Single Page Application (SPA).
- Clarifying concepts such as hash-based routing, IntersectionObserver, and DOM manipulation.
- Reviewing accessibility practices such as ARIA attributes and semantic HTML usage.
- Discussing performance optimization techniques identified through Lighthouse testing.
- Reviewing wording and grammar in written documentation.

---

## Examples of AI Assistance

### Example 1

Tool used: ChatGPT  
Purpose: Explanation of hash-based routing and URL fragments in JavaScript.  
Outcome: Improved understanding of how navigation between sections works without reloading the page.

### Example 2

Tool used: ChatGPT  
Purpose: Explanation of IntersectionObserver and how it keeps navigation state synchronized during scroll.  
Outcome: Implemented router logic that updates the URL hash and active navigation state while scrolling.

### Example 3

Tool used: ChatGPT  
Purpose: Review of Lighthouse performance feedback and discussion of how to optimize the Largest Contentful Paint (LCP) image.  
Outcome: Added image preload and priority hints to improve performance.

### Example 4

Tool used: ChatGPT  
Purpose: Reviewing and refining written explanations of the JavaScript modules used in the project.  
Outcome: Produced a clearer technical explanation document describing the architecture and behavior of the application.

AI assistance was used strictly as a learning and explanation tool and not as a substitute for understanding or completing the assignment independently.
